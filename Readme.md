## Pet-проект:
- **Telegram-бот** для интервального повторения (@TimeClickEducationBot)
Реализация системы интервального повторения по методу Эббингауза (концепция ANKI) для эффективного запоминания информации.
Бот учитывает кривую забывания: при правильных ответах увеличивает интервалы повторений, при ошибке возвращает к короткому интервалу.
Проект демонстрирует понимание когнитивных принципов запоминания и их практическое применение в разработке.
Стек: Go, Telegram Bot API, PostgreSQL, Redis
